关于听稿

产品介绍

听稿是专为内容创作者设计的在线文字转语音工具，帮助你通过"听"的方式审核稿件，提高审稿效率。

核心功能

  • 在线朗读：无需下载，支持多种语音
  • 选中位置朗读：从任意位置开始，无需从头听起
  • 完全免费：无需注册登录

技术说明

本工具基于 Web Speech API（W3C 开放标准）实现。所有语音合成在浏览器本地完成，文本内容不会上传到我们的服务器。

推荐使用 Microsoft Edge 浏览器以获得更自然的语音效果。

免责声明

  • 语音服务提供方：语音由您的浏览器提供（如 Edge、Chrome 等），我们仅调用浏览器 API
  • 服务质量：语音质量和可用性取决于浏览器，我们不对此承担责任
  • 用户责任：请勿输入违法、侵权或不当内容

知识产权

Web Speech API 是开放标准，可供任何网站免费使用。语音合成由浏览器厂商提供，相关权利归各厂商所有。本工具不存储、录制或分发任何语音文件。

隐私政策

最后更新：2026年5月

1. 信息收集

我们不收集您的文本内容。所有文字转语音处理都在您的浏览器本地完成，文本内容不会离开您的设备。

2. 本地存储

我们仅在您的浏览器本地保存以下设置（语音选择、语速/音调/音量、界面语言、输入框高度）。这些数据存储在您的设备上，我们无法访问。您可随时通过清除浏览器数据删除。

3. 第三方服务

重要：语音合成由您的浏览器提供（如 Edge、Chrome 等）。您的文本可能被发送到浏览器厂商的服务器进行处理，我们无法控制他们如何处理这些数据。

如关注隐私，请查阅：Microsoft 隐私声明 | Google 隐私政策

4. 数据安全

文本内容不会传输到我们的服务器。我们不使用 Cookies、不使用分析工具、不追踪用户行为、不收集个人信息、不与第三方共享数据。

5. 免责声明

本工具仅提供技术接口，实际语音服务由浏览器厂商提供。我们不对以下内容承担责任：

  • 浏览器厂商对您文本的处理方式
  • 语音合成的质量或可用性
  • 因使用本服务产生的任何损失
