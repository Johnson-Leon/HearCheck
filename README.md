# 听稿 HearCheck

一款极简的在线文字转语音工具，专为内容创作者设计，帮助你通过"听"的方式审核稿件，提高审稿效率。

## ✨ 核心功能

- **在线朗读**：无需下载，打开即用，支持多种中文和英文语音
- **上传文件**：支持导入 .txt 和 .md 文件（最大2MB）
- **选中位置朗读**：从任意位置开始朗读，无需从头听起
- **语音自定义**：可选择多个语音，调节语速、音调、音量
- **右键快捷操作**：右键点击语音标签快速移除
- **完全免费**：无需注册登录，无广告
- **隐私保护**：所有处理在浏览器本地完成，文本不上传服务器

## 🚀 快速开始

1. 访问 [HearCheck](https://hearcheck.407183993.workers.dev/)
2. 输入或上传要朗读的文本
3. 选择喜欢的语音
4. 点击"开始朗读"

## 💡 使用技巧

- **上传文件**：点击"上传文件"按钮，选择 .txt 或 .md 文件快速导入
- **从选定位置朗读**：在文本中点击想要开始的位置，会出现"从这开始朗读"按钮
- **管理语音**：点击"更多"按钮可以选择要显示的语音，右键点击语音标签可以快速移除
- **调节参数**：在"更多"设置中可以调节默认语速、音调、音量
- **切换语言**：右上角可以切换中英文界面

## 🎯 适用场景

- 📝 文章审稿：通过听的方式发现文字中的问题
- 📖 小说校对：长篇内容的错别字检查
- ✍️ 文案润色：听出语句是否通顺自然
- 🎓 学习辅助：听读学习材料，加深记忆
- 👁️ 视力保护：减少长时间盯屏幕的疲劳

## 🛠️ 技术栈

- **前端**：纯 HTML + CSS + JavaScript
- **语音合成**：Web Speech API (W3C 开放标准)
- **部署**：Cloudflare Workers

## 📋 浏览器支持

推荐使用 **Microsoft Edge** 浏览器以获得最佳语音效果。

其他支持的浏览器：
- Google Chrome
- Safari
- Firefox (部分功能)

## 🔒 隐私说明

- ✅ 文本内容不会上传到我们的服务器
- ✅ 所有语音合成在浏览器本地完成
- ✅ 不使用 Cookies 追踪用户
- ✅ 不收集个人信息
- ⚠️ 语音合成由浏览器厂商提供，文本可能被发送到浏览器厂商的服务器
