## 中文

一款极简的在线文字转语音工具，专为内容创作者设计，帮助你通过"听"的方式审核稿件，提高审稿效率。

### ✨ 核心功能

- **在线朗读**：无需下载，打开即用，支持多种中文和英文语音
- **上传文件**：支持导入 .txt 和 .md 文件（最大2MB）
- **选中位置朗读**：从任意位置开始朗读，无需从头听起
- **语音自定义**：可选择多个语音，调节语速、音调、音量
- **右键快捷操作**：右键点击语音标签快速移除
- **完全免费**：无需注册登录，无广告
- **隐私保护**：所有处理在浏览器本地完成，文本不上传服务器

### 🚀 快速开始

1. 访问 [HearCheck](https://hearcheck.407183993.workers.dev/)
2. 输入或上传要朗读的文本
3. 选择喜欢的语音
4. 点击"开始朗读"

### 💡 使用技巧

- **上传文件**：点击"上传文件"按钮，选择 .txt 或 .md 文件快速导入
- **从选定位置朗读**：在文本中点击想要开始的位置，会出现"从这开始朗读"按钮
- **管理语音**：点击"更多"按钮可以选择要显示的语音，右键点击语音标签可以快速移除
- **调节参数**：在"更多"设置中可以调节默认语速、音调、音量
- **切换语言**：右上角可以切换中英文界面

### 🎯 适用场景

- 📝 **文章审稿**：通过听的方式发现文字中的问题
- 📖 **小说校对**：长篇内容的错别字检查
- ✍️ **文案润色**：听出语句是否通顺自然
- 🎓 **学习辅助**：听读学习材料，加深记忆
- 👁️ **视力保护**：减少长时间盯屏幕的疲劳

### 🛠️ 技术栈

- **前端**：纯 HTML + CSS + JavaScript
- **语音合成**：Web Speech API (W3C 开放标准)
- **部署**：Cloudflare Workers

## English

A minimalist online text-to-speech tool designed for content creators to review drafts by listening, improving proofreading efficiency.

### ✨ Core Features

- **Online Reading**: No download required, ready to use, supports multiple Chinese and English voices
- **File Upload**: Import .txt and .md files (max 2MB)
- **Start from Selection**: Begin reading from any position, no need to start over
- **Voice Customization**: Choose multiple voices, adjust speed, pitch, and volume
- **Right-click Shortcuts**: Right-click voice tags for quick removal
- **Completely Free**: No registration required, no ads
- **Privacy Protection**: All processing done locally in browser, text not uploaded to servers

### 🚀 Quick Start

1. Visit [HearCheck](https://hearcheck.407183993.workers.dev/)
2. Enter or upload text to read
3. Select your preferred voice
4. Click "Start"

### 💡 Usage Tips

- **Upload Files**: Click "Upload" button to quickly import .txt or .md files
- **Start from Selection**: Click anywhere in the text to see "Start from here" button
- **Manage Voices**: Click "More" to select voices to display, right-click voice tags to remove
- **Adjust Parameters**: Configure default speed, pitch, and volume in "More" settings
- **Switch Language**: Toggle between Chinese and English interface in top-right corner

### 🎯 Use Cases

- 📝 **Article Review**: Discover text issues by listening
- 📖 **Novel Proofreading**: Check typos in long content
- ✍️ **Copy Polishing**: Hear if sentences flow naturally
- 🎓 **Learning Aid**: Listen to study materials for better retention
- 👁️ **Eye Protection**: Reduce screen time fatigue

### 🛠️ Tech Stack

- **Frontend**: Pure HTML + CSS + JavaScript
- **Speech Synthesis**: Web Speech API (W3C open standard)
- **Deployment**: Cloudflare Workers

### 📋 Browser Support

**Microsoft Edge** browser is recommended for the best voice quality.

Other supported browsers:
- Google Chrome
- Safari
- Firefox (partial features)

### 🔒 Privacy Notice

- ✅ Text content is not uploaded to our servers
- ✅ All speech synthesis is done locally in your browser
- ✅ No Cookies used to track users
- ✅ No personal information collected
- ⚠️ Speech synthesis is provided by browser vendors, text may be sent to their servers

See [Privacy Policy](https://hearcheck.407183993.workers.dev/) for details
