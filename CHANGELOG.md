# 更新日志 Changelog

## [1.0.0] - 2026-05-26

### 新增功能 ✨
- **文件上传功能**：支持上传 .txt 和 .md 文件（最大2MB）
- **右键菜单**：右键点击语音标签可快速移除
- **覆盖确认**：上传文件时如果输入框有内容会提示确认
- **文件大小限制**：限制上传文件最大2MB，防止浏览器卡顿
- **完整的按钮状态管理**：所有按钮根据内容状态智能启用/禁用

### 优化改进 🎨
- **统一配色方案**：全站采用黑白灰配色，更加简洁专业
- **按钮布局优化**：调整为"上传文件 | 开始朗读 | 停止" / "复制全部 | 清空"
- **按钮命名优化**：
  - "复制" → "复制全部"（更准确）
  - "导入" → "上传文件" → "Upload"（更简洁）
- **Toast 提示统一**：所有操作都有相应的成功/失败提示
- **右键菜单样式**：采用黑白灰配色，悬停效果更自然

### 功能完善 🔧
- **删除声音逻辑优化**：
  - 可以删除所有声音（包括默认声音）
  - 删除后不会自动恢复默认声音
  - 删除后显示成功提示
- **语言切换优化**：
  - 保持无缝切换，不弹出确认框
  - 右键菜单文本随语言切换
- **按钮状态完善**：
  - 清空按钮：有内容时可用，无内容时禁用
  - 开始朗读按钮：有内容时可用，无内容时禁用
  - 复制全部按钮：有内容时可用，无内容时禁用
  - 上传文件按钮：始终可用

### 用户体验 💡
- **清空提示**：点击清空按钮后显示"✅ 已清空"提示
- **文件格式提示**：上传不支持的格式时显示明确提示
- **文件过大提示**：上传超过2MB的文件时显示提示
- **覆盖确认对话框**：上传文件前如果有内容会询问是否覆盖

### 文档更新 📝
- 完善 README.md，添加详细的功能介绍和使用说明
- 更新 sitemap.xml 日期
- 添加 CHANGELOG.md 记录更新历史

### 技术细节 🛠️
- 文件读取使用 UTF-8 编码，支持中文
- 文件大小检查：2 * 1024 * 1024 bytes (2MB)
- 支持的文件格式：.txt, .md
- 右键菜单使用原生 contextmenu 事件
